Products  
ProductID – jedinstveni broj proizvoda generisan od strane sistema  
Uid – jedinsteni guid proizvoda  
Caption  - kratak naziv proizvoda  
Description – Opis proizvoda  
Price – Cena proizvoda na dve decimale  
SizeEID – velicina proizvoda, strani kljuc na lookup tabelu SizeTypes  
BrandEID – proizvodjac, strani kljuc na lookup tabelu BrandTypes 
 
Categories  
CategoryID – id kategorije  
Uid – jedinsteni guid kategorije  
Caption – kratak naziv kategorije 

SizeTypes  
SizeTypeEID – id velicine  
Caption – kratak naziv velicine 

BrandTypes  
BrandTypeEID – id proizvodjaca  
Caption – kratak naziv proizvodjaca
